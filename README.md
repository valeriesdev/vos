Valerie OS                           {#mainpage}
==========

An operating system built over quite some time, largely based off of cfenollosa's tutorials.
## Features
 - [X] Bootloader
 - [X] Graphics Driver
 - [X] Keyboard Driver
 - [X] Memory Allocator
 - [X] Expandable C Kernel
 - [X] Hard Disk Reading
 - [ ] Modular interface for programs
 - [ ] C Compiler
 - [ ] Text Editor (in progress)
 - [X] File System 
 - [ ] Convert from makefile to CMake
